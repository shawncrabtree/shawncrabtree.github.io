The Honors College uses a web based application for tracking the entire student lifecycle. Cambia Care Coordination

My team at Cambia created a large scale project using health insurance records to implement gains share, care coordination fees and attributions to provider networks in many states around the country. Smarter Balanced Assessment Consortium is a company that provides electronic standardized tests to school systems. I developed an AngularJS proof of concept for adding glossary images to the tests. I also created a C# console application to add these glossary images to the project metadata.
